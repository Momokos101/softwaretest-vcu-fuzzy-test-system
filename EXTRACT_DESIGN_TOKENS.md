# 从Figma Dev Mode提取设计规范 - 快速指南

## 🎯 目标

使用Figma Dev Mode提取设计规范，然后我会帮你创建配置文件并整合到项目中。

## 📋 提取步骤（5分钟完成）

### 步骤1: 切换到Dev Mode

1. 在Figma中打开你的设计文件
2. 点击右上角的 **"Dev Mode"** 按钮
3. 界面会切换到开发模式

### 步骤2: 提取颜色（2分钟）

1. **选择主色调**
   - 点击设计中的主要按钮或强调色
   - 在右侧面板查看"Fill"部分
   - 记录颜色值（HEX格式）

2. **选择其他颜色**
   - 文字颜色（主、次）
   - 背景色
   - 边框色
   - 状态色（成功、警告、错误）

**记录格式**：
```
主色调: #1890ff
辅助色: #52c41a
文字主色: #000000
文字次色: #666666
背景色: #ffffff
边框色: #d9d9d9
成功色: #52c41a
警告色: #faad14
错误色: #ff4d4f
```

### 步骤3: 提取字体（1分钟）

1. **选择标题文本**
   - 点击H1标题
   - 记录：字号、字重、行高

2. **选择正文文本**
   - 点击正文
   - 记录：字号、字重、行高

**记录格式**：
```
H1: 24px, 700, 1.2
H2: 20px, 600, 1.3
正文: 14px, 400, 1.5
小字: 12px, 400, 1.4
字体族: -apple-system, BlinkMacSystemFont, "Segoe UI"
```

### 步骤4: 提取间距（1分钟）

1. **查看元素间距**
   - 选择两个相邻元素
   - 查看它们之间的间距
   - 记录常用间距值

**记录格式**：
```
常用间距: 4px, 8px, 12px, 16px, 24px, 32px
```

### 步骤5: 提取组件样式（1分钟）

1. **选择按钮**
   - 记录：圆角、内边距、高度

2. **选择输入框**
   - 记录：圆角、内边距、高度

3. **选择卡片**
   - 记录：圆角、内边距、阴影

**记录格式**：
```
按钮圆角: 4px
按钮内边距: 0 16px
按钮高度: 32px
输入框圆角: 4px
卡片圆角: 8px
卡片内边距: 24px
```
