# Figma Dev Mode 使用指南（专业版）

## 🎯 使用Dev Mode的优势

作为Figma专业版用户，你可以使用**Dev Mode**，这是最准确和高效的方法：
- ✅ 直接查看CSS属性
- ✅ 自动生成设计规范
- ✅ 精确的间距和尺寸
- ✅ 颜色、字体等设计token
- ✅ 可以直接复制代码

## 🚀 快速开始

### 步骤1: 切换到Dev Mode

1. 在Figma中打开你的设计文件
2. 点击右上角的 **"Dev Mode"** 按钮（或使用快捷键）
3. 界面会切换到开发模式，显示代码相关信息

### 步骤2: 提取设计规范

#### A. 提取颜色（Design Tokens）

1. **选择任意元素**
   - 点击设计中的任意元素

2. **查看颜色值**
   - 在右侧面板的"Fill"部分
   - 可以看到HEX、RGB、HSL等格式
   - 点击颜色值可以复制

3. **记录所有颜色**
   - 主色调
   - 辅助色
   - 文字颜色
   - 背景色
   - 边框色
   - 状态色（成功、警告、错误等）

**示例输出**：
```
主色调: #1890ff
辅助色: #52c41a
文字主色: #000000
文字次色: #666666
背景色: #ffffff
边框色: #d9d9d9
```

#### B. 提取字体规范

1. **选择文本元素**
   - 点击设计中的文本

2. **查看字体信息**
   - 字体族（Font Family）
   - 字号（Font Size）
   - 行高（Line Height）
   - 字重（Font Weight）
   - 字间距（Letter Spacing）

3. **记录所有字体样式**
   - 标题字体（H1, H2, H3等）
   - 正文字体
   - 小字体
   - 按钮字体

**示例输出**：
```
H1: 24px, 700, 1.2
H2: 20px, 600, 1.3
Body: 14px, 400, 1.5
Small: 12px, 400, 1.4
```

#### C. 提取间距系统

1. **使用测量工具**
   - 在Dev Mode中，选择元素
   - 查看元素之间的间距
   - 记录常用间距值

2. **记录间距规范**
   - 常用间距：4px, 8px, 12px, 16px, 24px, 32px等

#### D. 提取组件样式

1. **选择组件**
   - 点击按钮、输入框等组件

2. **查看CSS属性**
   - 在右侧面板可以看到CSS代码
   - 可以直接复制

3. **记录组件样式**
   - 按钮：圆角、内边距、阴影等
   - 输入框：边框、圆角、焦点状态等
   - 卡片：圆角、阴影、内边距等

### 步骤3: 导出设计规范

1. **使用Figma的Design Tokens功能**
   - 在Dev Mode中可以导出设计规范
   - 包括颜色、字体、间距等

2. **或者手动记录**
   - 创建一个文档记录所有规范
   - 我会帮你转换成代码

## 📝 实际操作步骤

### 方法1: 使用Dev Mode直接复制CSS

1. 在Dev Mode中选择元素
2. 右侧面板会显示CSS代码
3. 直接复制CSS
4. 保存到项目样式文件中

### 方法2: 导出Design Tokens

1. 在Figma中，点击右上角菜单
2. 选择"Export" → "Design Tokens"
3. 选择导出格式（JSON/CSS）
4. 保存到项目

### 方法3: 手动记录（最简单）

1. 打开一个文本编辑器
2. 按照上面的清单，逐个记录设计规范
3. 发给我，我会帮你转换成代码

## 🎨 示例：提取设计规范

假设你提取了以下信息：

**颜色**：
- 主色: #1890ff
- 成功: #52c41a
- 警告: #faad14
- 错误: #ff4d4f

**字体**：
- 字体族: -apple-system, BlinkMacSystemFont, "Segoe UI"
- H1: 24px, 700
- Body: 14px, 400

**间距**：
- 小: 8px
- 中: 16px
- 大: 24px

我会帮你创建：

```javascript
// colors.js
export const colors = {
  primary: '#1890ff',
  success: '#52c41a',
  warning: '#faad14',
  error: '#ff4d4f',
}

// typography.js
export const typography = {
  fontFamily: '-apple-system, BlinkMacSystemFont, "Segoe UI"',
  h1: { fontSize: '24px', fontWeight: 700 },
  body: { fontSize: '14px', fontWeight: 400 },
}

// spacing.js
export const spacing = {
  small: '8px',
  medium: '16px',
  large: '24px',
}
```
