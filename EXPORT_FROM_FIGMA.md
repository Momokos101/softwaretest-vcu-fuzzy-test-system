# 从Figma导出代码 - 完整步骤

## 🚀 快速导出方法

### 方法1: 使用Figma插件导出（推荐）

#### 步骤1: 安装插件

1. 在Figma中打开你的设计文件
2. 点击左侧菜单的 **"插件"** 图标（或按 `Cmd/Ctrl + /`）
3. 搜索以下插件之一并安装：

**推荐插件**：
- **Figma to Code** (最常用，免费)
- **Anima** (功能强大)
- **Figma to React** (专门针对React)

#### 步骤2: 导出代码

1. **选择要导出的页面/组件**
   - 可以全选整个页面
   - 或选择特定组件

2. **运行插件**
   - 点击插件图标启动
   - 选择导出格式：**React**
   - 选择代码风格：**函数式组件** (Function Component)

3. **复制代码**
   - 插件会生成代码
   - 复制所有生成的代码
   - 保存到临时文件或直接发给我

### 方法2: 使用Dev Mode查看CSS

1. **切换到Dev Mode**
   - 点击右上角 "Dev Mode" 按钮

2. **选择元素查看CSS**
   - 选择任意元素
   - 右侧面板会显示CSS代码
   - 可以复制CSS

3. **导出设计规范**
   - 在Dev Mode中可以导出Design Tokens
   - 包括颜色、字体、间距等

## 📋 导出清单

导出时请注意：

- [ ] 导出所有页面组件
- [ ] 导出所有可复用组件
- [ ] 导出样式文件
- [ ] 记录设计规范（颜色、字体、间距）

## 🔧 导出后的处理

导出代码后，我会帮你：

1. ✅ 调整代码结构以符合项目
2. ✅ 修复导入路径
3. ✅ 整合到现有项目结构
4. ✅ 连接后端API
5. ✅ 确保代码可以运行

## 📝 导出代码的格式

导出后，代码可能是这样的格式：

```jsx
// 组件代码
function MyComponent() {
  return (
    <div className="container">
      <h1>标题</h1>
    </div>
  )
}

// CSS代码
.container {
  padding: 24px;
}
```

我会帮你：
- 将组件放到 `frontend/src/components/` 或 `frontend/src/pages/`
- 将样式放到 `frontend/src/styles/`
- 调整导入路径
- 连接API服务
